Para trabaajar con git use o Anaconda Prompt o GitBash

En ambos casos hay que definir donde vas a crear el repositorio con
cd "Direccion/Donde/Crear/Repositorio"

Para generar repositorio:
git clone https://github.com/dandrey-ai/AnalisisVegetacionFAO.git

Para hacer cambios:
git add .

Para hacer comentarios:
git commit

Para llevar los cambios que hice desde el repositorio clonado a git:
git push

Para bajar los cambios que hicieron los demas a mi copia del repositorio:
git pull
